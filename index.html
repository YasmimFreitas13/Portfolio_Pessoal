Projetos

Qualidade do Ar

O Qualidade do Ar foi desenvolvido durante um desafio da FIAP + Alura, com foco em HTML, CSS, JavaScript e Python (Flask). O objetivo foi criar uma aplicação que informa a qualidade do ar da cidade digitada, consumindo dados de uma API externa e classificando os poluentes em Ótimo, Regular ou Ruim. É um ótimo exercício para praticar consumo de API, full stack e deploy em nuvem com Render.

Spotify

O Clone de Interface do Spotify foi desenvolvido durante a Imersão da Alura com foco em HTML, CSS e JavaScript. O objetivo foi recriar visualmente a interface do Spotify Web, praticando conceitos de layout responsivo com Flexbox e Grid, uso de variáveis CSS e organização semântica. É um ótimo exercício para treinar construção de interfaces modernas e atenção aos detalhes de design.

Jogo do Número Secreto

O Jogo do Número Secreto é um projeto simples desenvolvido durante um curso da Alura, com o objetivo de praticar HTML, CSS e JavaScript. No jogo, o sistema sorteia um número aleatório e o jogador tenta adivinhar, recebendo dicas se o número é maior ou menor. O projeto ensina conceitos como manipulação do DOM, eventos, lógica condicional e geração de números aleatórios.
